# Практическая работа №6

### Инструкция
1. Клонируем проект с файлами 
    ```shell
    git clone https://github.com/luminous453/prac6_1
   ```
2. Устанавливаем зависимости и тд
    ```shell
    npm install
   ```
3. Переходим в папку session-auth
   ```shell
   cd session-auth
   ``` 
5. Запускаем сервер 
    ```shell
    node server.js
   ```

6. Переходим на сайт по ссылке
   ```shell
    http://localhost:3000/
   ```
